---

## 1. 회의한 내용

- 금일 윤상이는 개인 작업 종료 후,
  오후에 **모든 작업물을 전체 검토**하고 수정이 필요한 부분을 정리하기로 함
- CSS 선택자 관련 논의
  - `class` 선택자 사용 부분을 → `id` 선택자로 변경 필요
- 작업물 검토 중 공통 스타일 개선 필요 사항 도출
  - 페이지별 **바탕색 통일**
  - **글씨 두께(font-weight) 통일**
  - **요소 모서리(border-radius) 통일**
- 위 항목들을 중심으로 전반적인 스타일 정리를 진행하기로 함

## 5. 회고록

- **윤상이**
  - 작업 실수로 인한 데이터 손실을 복구하면서
    저장과 백업의 중요성을 다시 한 번 인식함
  - 팀원들의 도움으로 빠르게 복구가 이루어졌고,
    남은 작업들도 팀원들이 정리 및 작성해 주어
    큰 문제 없이 프로젝트를 이어갈 수 있었음
  - 작업 이후 강사님의 피드백을 반영하며
    기존 작업물을 다시 검토해 보니 부족한 부분이 많다고 느껴
    다음부터는 더 정확하고 신중하게 작업하겠다고 다짐함

- **현길용**
  - 인라인 스타일과 인라인 스크립트는 사용하면 안 된다는 점을 인식함
  - 스크립트 적용 시 `defer`의 중요성을 알게 됨
  - `DOMContentLoaded` 이벤트를 잊지 말아야 함
  - `class`보다 `id` 선택자를 사용하는 것이 적절하다는 점을 정리함

- **최서윤**
  - GitHub를 활용해 팀원들의 작업물을 공유하고
    수정하는 과정을 반복하며 이전보다 능숙해졌음을 느낌
  - 작업 과정에서 협업의 효율성과 성취감을 경험함
  - 지금까지의 작업물을 검토하며
    CSS와 JavaScript에 대한 이해가 아직 부족하다고 느껴
    추가 학습의 필요성을 인식함
